Count Sort

Count sort is an efficient sorting algorithm that is particularly useful when dealing with a known range of integer values. It works by counting the occurrences of each element in the input array and using this information to place the elements in the correct position.

Description

  • Counting Occurrences: Count sort creates a count array where each index represents an element in the input array.
  • Placing Elements: After counting, the count array is used to place the elements in the correct sorted order.
  • Efficient for Small Range: Count sort is efficient when the range of input values is small.

Example

Consider sorting the array [4, 2, 2, 8, 3, 3, 1] using count sort:

  1. Step 1: Create a count array based on the input array:
  2. [0, 1, 2, 2, 1, 0, 0, 1, 1]
  3. Step 2: Use the count array to place elements in sorted order:
  4. [1, 2, 2, 3, 3, 4, 8]

Characteristics

  • Time Complexity: Count sort has a time complexity of O(n + k), where n is the number of elements and k is the range of input values.
  • Space Complexity: O(k), since the count array requires additional space proportional to the range of input values.
  • Stable: Count sort is stable, meaning it preserves the relative order of equal elements.
  • Not In-Place: It requires additional space for the count array, so it is not an in-place sorting algorithm.

Count sort is especially useful for sorting integers within a known range, but it may not be as efficient for larger ranges or non-integer data.
