Sorting Algorithms

Sorting is a fundamental operation in computer science, used to arrange data in a specific order. There are various algorithms for sorting, each with different efficiencies in terms of time and space. Here are some commonly used sorting algorithms:

  • Insertion Sort
  • Bubble Sort
  • Selection Sort
  • Merge Sort
  • Quick Sort
  • Count Sort
  • Radix Sort

Insertion Sort

Insertion sort is a simple, in-place, comparison-based algorithm that sorts a list by inserting elements into their correct position within a growing sorted portion.

Learn more about Insertion Sort

Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order, iterating until the list is sorted.
