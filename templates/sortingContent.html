Sorting

Sorting is a technique which sorts the data. There are many techniques which sort data with diferent time complexity and with different space complexity. Some techniques are written below:

  • Insertion Sort
  • Bubble Sort
  • Selection Sort
  • Merge Sort

  • Radix Sort
Insertion Sort
Insertion sort is a simple, in-place, comparison-based algorithm that sorts a list by inserting elements into their correct position within a growing sorted portion.
Details: Insertion Sort
Bubble Sort
Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order, iterating until the list is sorted.
Craving for details Bubble Sort
Selection Sort
Selection sort finds the minimum element from the unsorted portion and swaps it with the first unsorted element, gradually building a sorted array.
Details: Selection Sort
Merge Sort
Merge sort is a divide-and-conquer algorithm that recursively splits the array, sorts each half, and then merges the sorted halves to produce a sorted array.
Details: Merge Sort
Quick Sort
Quick sort is a divide-and-conquer algorithm that selects a pivot element, partitions the array around the pivot, and recursively sorts the subarrays.
Details: Quick Sort
Count Sort
Count sort is an integer sorting algorithm that counts occurrences of each value to place values in correct positions in a sorted array.
Details: Count Sort
Radix Sort
Radix sort is a non-comparative algorithm that sorts integers by processing individual digits from least significant to most significant.
Details: Radix Sort
