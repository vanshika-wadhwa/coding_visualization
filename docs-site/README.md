# Coding Visualization Platform

A web-based learning platform that helps students understand **Data Structures and Algorithms** through interactive visualizations.

Built using Flask and SQLite, the platform provides visual demonstrations of algorithms and data structures such as stacks, queues, linked lists, trees, and sorting algorithms.

## Features

* User authentication system (Register / Login / Logout)
* Password reset using email
* Visualization of sorting algorithms
* Data structure learning modules
* Practice session environment
* Interactive learning dashboard

## Installation

Clone the repository

git clone https://github.com/vanshika-wadhwa/coding_visualization.git

Move into the project folder

cd coding_visualization

Create virtual environment

python -m venv venv

Activate environment

Windows
venv\Scripts\activate

Install dependencies

pip install flask flask_sqlalchemy flask_mail

Run the application

python app.py

Open browser

http://localhost:8000
