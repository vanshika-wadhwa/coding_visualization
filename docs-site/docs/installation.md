# Installation Guide

Follow these steps to run the Coding Visualization Platform locally.

## 1. Clone the repository
git clone https://github.com/yourusername/coding_visualization.git

## 2. Navigate to the project
cd coding_visualization

## 3. Create a virtual environment
python -m venv venv


## 4. Activate the environment

Windows:
venv\Scripts\activate

## 5. Install dependencies
pip install flask flask_sqlalchemy flask_mail

## 6. Run the application
python app.py
Open browser:
http://localhost:8000
