# Coding Visualization Platform

Welcome to the documentation of the **Coding Visualization Platform**.

This project is a web-based learning system designed to help students understand **Data Structures and Algorithms** using interactive visualizations.

The platform is built using Python and Flask and provides visual demonstrations of algorithms such as sorting and data structures like stacks, queues, linked lists, and trees.

## Key Features

- Interactive algorithm visualizations
- Learning modules for data structures
- Sorting algorithm demonstrations
- User authentication system
- Practice session environment

## Technologies Used

Backend:
- Python
- Flask
- Flask-SQLAlchemy
- Flask-Mail

Frontend:
- HTML
- CSS
- JavaScript

Database:
- SQLite
